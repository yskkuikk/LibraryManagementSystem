# Library-Management-System
A java based dekstop application that automates library operations such as user authentication, book management, member tracking and borrowing/returning books using PostgreSQLdatabase.
## Team members and roles
Damilya - Java programming and database development
Ilona - Java programming and system architecture
## Project Goal
The goal of this project is to create a logical and secure system that is efficient unlike manual library record management. Our project will automate core library operations while ensuring accurate tracking of users and borrowed materials.
## MVP Features
-User Authentication
-Member and User management
-Book and Inventory management
-Borrowing, Returning and Due-Date Management
-Activity logging and history tracking
-Error handling and Data validation
User login (Member / Librarian)
## Technologies used
- JAVA (JDK 25)
- PostgreSQL 18
- JDBC
- pgAdmin
- Intellij IDEA
- Github
